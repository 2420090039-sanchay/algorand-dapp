\# Algorand Smart Contract Demo



\## Description



This project demonstrates a basic smart contract application built on the Algorand blockchain. It showcases how to create, deploy, and interact with smart contracts using AlgoKit and the Algorand JavaScript SDK.



\## Features



\* Create and deploy a simple smart contract

\* Interact with the contract using transactions

\* Local development using Algorand LocalNet

\* Wallet integration using Pera Wallet



\## Tech Stack



\* Algorand Blockchain

\* AlgoKit

\* JavaScript (Algorand JS SDK)

\* Node.js



\## Setup Instructions



1\. Clone the repository

2\. Install dependencies using npm install

3\. Start LocalNet using AlgoKit

4\. Deploy the smart contract

5\. Run the application
